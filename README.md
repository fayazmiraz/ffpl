# FFPL (FireFox Profile Launcher)

A CLI tool to launch Firefox with a selected profile name.

## Installation

```sh
sudo i ffpl

# to access globally in Linux, MacOS
sudo npm i -g ffpl
```

## Usage

Run the following command to launch Firefox with a selected profile:

```sh
ffpl
```
